### 局部安装并使用 vue-cli(或者其他 npm 包)

1. 新建目录：`mkdir vue-cli2`

2. 切换目录：`cd vue-cli2`

3. 初始化项目：`npm init -y`

4. 安装：`npm i vue-cli`

5. 查看版本：`npx vue -V`

6. 使用：`npx vue init webpack project`

- 补充：

1. 安装 `vue-cli3.x：npm i @vue/cli@3`

7. 使用：`npx vue create project`
   
8. 或者 `npx vue ui` 打开可视化界面创建 project
